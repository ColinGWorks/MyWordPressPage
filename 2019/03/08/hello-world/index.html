Hello world!

Now it is 2019, and it gets pretty easy with setting up a web page managed by WordPress, hosted on highly available servers with minimum cost. On Mar 1st, Google has released the .dev TLD (top level domain). Hence I found this www.devwave.dev domain name for my use, of cause with a small amount of registration fee.

Here is a brief history of the .dev domain names. All TLD names such as .com, .edu are managed by organization named ICANN. At around 2009, it decided to open up TLDs to any names, as long as a determined register fees are paid. Thus, .engineering, .news, .app, and .club, all these novel names become TLDs. And Google has applied to own the TLDs such as .dev and .chrome. Previously, developers has been using .dev domain names as an alias to localhost, and use for example www.standard.dev as URLs for testing locally hosted web services. After Google become the owner of .dev TLD and now release them for public use, this practice has recently become outdated. But still, Google intends public to register .dev domain names for hosting web sites that are dedicated to development and technology.
